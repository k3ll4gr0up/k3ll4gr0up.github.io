- Lunar Lander -

Lunar Lander is a space game written in C for my school.

The main goal of Lunar Lander is to land a spaceship on the moon. But wait. It's not graphical at all. in fact, it's a text based game, which prints out a table. In there you can get the current values of height, speed and fuel. You have to decide how much fuel you burn every ten seconds.

Lunar Lander was originally written by a stundent called Jim Storer in 1969. It was one of the first ever computer games.

The original game was written in FOCAL, a interpreted programming language. FOCAL is an abbreviation of Formula Calculator.

- Coffee -

Coffee is a command line interface for the RFC2324 which is a communications protocol for controlling, monitoring and fiagnosing a coffee pot.

The main goal of this project is to create a working implementation of this protocol.

We plan to control a Senseo® machine using a ATMEL microcontroller for acessing the temperature data and brew the selected coffee. There will be a CLI, an App and a web interface available when we're finished.

The RFC2324 is a facetious communications protocol for controlling, monitoring, and diagnosing coffee pots. It's called the Hyper Text Coffee Pot Control Protocol.

This project will definitely make your coffee machine more awesome than it is right now.

- marcopolo -

marcopolo is a simple client-server application written in C.

The original purpose of this application was that a client could broadcast "marco" into the network and every server replies with "polo". The client application would then print a list of every polo recieved.

Since I was unable to get this broadcast thing working, this is just a very simple ping-like application.
